# HowText 🌈

[![PyPI version](https://img.shields.io/pypi/v/HowText.svg)](https://pypi.org/project/HowText/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.6+](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/downloads/)

**HowText** is a high-performance, real-time rainbow text animation engine for Python terminals. It uniquely solves the problem of static-colored emojis by intelligently mapping them to color-flowable Unicode symbols.



---

## ✨ Key Features

*   **⚡ Real-time RGB Animation:** Smooth rainbow transitions using a high-frequency Sine-wave algorithm.
*   **🧠 Smart Emoji Mapping:** Automatically detects and converts standard emojis into dynamic Unicode counterparts.
*   **🚀 Ultra Lightweight:** Zero external dependencies (Pure Python) to ensure maximum compatibility.
*   **🛠 Developer Friendly:** Simple API designed for quick integration into CLI tools and bots.
